XZMediaPlayer 是 XZKit 组件，目前处于初步开发阶段，该组件的主要目的是为 iOS App 提供原生、简单、高效的视频播放器。

## 开发计划：

- 阶段 1: 实现原生视频播放，全屏窗口播放，自定义控制 UI 。
- 阶段 2: 实现缓存、列队播放。
- 阶段 3: 实现音频播放，尝试集成 ffmpeg 。
